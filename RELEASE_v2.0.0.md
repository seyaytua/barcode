# リリース v2.0.0

## 📅 リリース日
2024年12月9日

## 🎉 概要
バーコード/QRコード台紙作成システムのメジャーバージョン2.0.0をリリースしました。
main_v2.pyをベースとした新しいアーキテクチャにより、より安定した動作とメンテナンス性の向上を実現しました。

## ✨ 主な特徴

### バーコード/QRコード台紙作成システム
- **備品用バーコード/QRコード**: サイズ可変の台紙作成（サイズ1-8対応）
  - サイズ1-2: 8列レイアウト
  - サイズ3-4: 6列レイアウト
  - サイズ5-6: 4列レイアウト
  - サイズ7-8: 2列レイアウト
- **教員用バーコード/QRコード**: 2列レイアウトのコンパクトな台紙作成
- **Excelインポート**: テンプレートを使用した簡単なデータ入力
- **プレビュー機能**: PDF出力前に台紙のレイアウトを確認可能
- **日付設定**: 作成日付を自由に指定可能
- **フォントサイズ調整**: 番号表示のサイズをカスタマイズ可能（8-20pt）

## 🔧 技術仕様

### 対応プラットフォーム
- Windows 10/11 (64-bit)
- macOS (Intel/Apple Silicon)

### 技術スタック
- **Python**: 3.11+
- **GUI フレームワーク**: PySide6
- **バーコード生成**: python-barcode
- **QRコード生成**: qrcode
- **PDF処理**: ReportLab
- **Excel処理**: openpyxl
- **画像処理**: Pillow

### ビルド情報
- PyInstaller使用によるスタンドアロン実行ファイル
- 依存関係を含む単一実行ファイル形式
- Windows Defenderによる誤検知に対応済み

## 📦 ダウンロード

### Windows版
- `BarcodeQRCodePDF_v2-Windows-v2.0.0.zip` をダウンロード
- ZIPファイルを展開
- `BarcodeQRCodePDF_v2.exe` を実行

### macOS版
- `BarcodeQRCodePDF_v2-macOS-v2.0.0.dmg` または `.zip` をダウンロード
- DMGファイルをマウントしてアプリケーションフォルダにドラッグ
- 初回起動時は右クリック → 「開く」を選択

## 🚀 使用方法

### 基本的な使用手順

1. **設定**
   - コードタイプを選択（バーコード/QRコード）
   - 作成日付を指定
   - 番号サイズを調整（必要に応じて）

2. **データ入力**
   - 「📥 テンプレート出力」ボタンでExcelテンプレートを出力
   - Excelで必要なデータを入力
     - 備品用: 対象名、コード番号、サイズ、備考
     - 教員用: 教員名、コード番号、備考
   - 「📂 Excel読み込み」ボタンで入力したExcelを読み込み

3. **プレビュー**
   - 「👁️ 備品プレビュー」または「👁️ 教員プレビュー」でレイアウトを確認
   - ページ送り機能で全ページを確認可能

4. **PDF出力**
   - 「🖨️ 備品PDF出力」または「🖨️ 教員PDF出力」でPDFファイルを生成
   - 保存先を指定して完了

## 🐛 既知の問題と対処法

### Windows Defender警告
- Windows Defenderが警告を表示する場合があります
- 対処法: 「詳細情報」をクリックして「実行」を選択してください
- これはPyInstallerでビルドされた実行ファイルによくある誤検知です

### macOS Gatekeeper警告
- 初回起動時にmacOSのセキュリティ警告が表示されます
- 対処法: 右クリック → 「開く」を選択してください

### 起動しない場合
- 管理者権限で実行してみてください
- アンチウイルスソフトが干渉している可能性があります
