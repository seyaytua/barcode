# リリースノート v2.0.0

## 概要
バーコード/QRコードPDF印刷システム v2.0.0のリリースです。

## 変更内容

### 新機能
- main_v2.pyベースの新しいアプリケーション
- バーコード管理システムの統合UI
- 教職員と備品のタブ管理機能
- 日付指定機能の追加
- Excelエクスポート/インポート機能の強化

### ビルド情報
- ビルド対象: main_v2.py
- 実行ファイル名: BarcodeQRCodePDF_v2
- バージョン: 2.0.0
- ビルド日: 2024-12-09

### 技術スタック
- Python 3.11+
- PySide6 6.6.0
- python-barcode 0.15.1
- qrcode 7.4.2
- Pillow 10.1.0
- PyPDF2 3.0.1
- reportlab 4.0.7
- openpyxl 3.1.2
- PyMuPDF 1.23.8

### ダウンロード
実行ファイルは `dist/BarcodeQRCodePDF_v2` から入手できます。

### 使用方法
1. 実行ファイルをダウンロード
2. Windows環境で実行
3. アプリケーションが起動します

### システム要件
- Windows 10以降
- メモリ: 2GB以上推奨
- ディスク空き容量: 200MB以上

### 既知の問題
- なし
